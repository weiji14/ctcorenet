# CTCoreNet

Neural network for classifying rock clasts in computed tomography (CT) scans of sediment cores.

# Getting started

## Quickstart

Launch in [Pangeo Binder](https://pangeo-binder.readthedocs.io) (Interactive jupyter lab environment in the cloud).
